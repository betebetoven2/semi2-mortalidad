# Fase 3 — Explotación Analítica, Machine Learning y BI

## Alcance

Esta sección documenta la **Fase 3** del proyecto: explotación analítica sobre el repositorio dimensional construido en la Fase 2. El foco ya no está en la conformación de datos, sino en el uso analítico del Data Warehouse para producir evidencia, visualización e interoperabilidad.

La documentación se organiza en cuatro bloques:

1. **Modelo de aprendizaje automático**
2. **Visualización analítica e interoperabilidad BI**
3. **Análisis comparativo y recomendaciones**
4. **Entregables técnicos y criterios de aceptación**

## Ubicación dentro del proyecto

- **Fase 1:** ingesta, Sandbox y gobernanza inicial
- **Fase 2:** Stage y Data Warehouse
- **Fase 3:** explotación analítica, aprendizaje automático y consumo BI

## Flujo de referencia

```text
Sandbox → Stage → Data Warehouse → Modelo de ML → Visualizaciones BI → Recomendaciones
```

## Evidencia utilizada

La documentación de esta fase se apoya en:

- el informe final de consultoría,
- los cuadernos exportados desde Databricks,
- y el enunciado oficial de la asignatura para la Fase 3.

## Criterio rector

La fase debe demostrar que el repositorio analítico no solo almacena datos, sino que también permite construir modelos predictivos, soportar tableros en herramientas distintas y traducir los hallazgos en recomendaciones de política pública.
